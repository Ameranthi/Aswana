Aswana version (Beta) 2.0. Bugs patched out, text made compatible with windows. Not all drawings done. 

To run this game:

* Download Java & JGRASP (if you use some other IDE, best of luck getting it to work)
* Put the folder Aswana on the desktop
* In JGRASP, in file directory (left hand side of window), find Aswana 
* Click the following sequence: Aswana -> src -> project -> ProjectDemo.java
* compile then run file
